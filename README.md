# Request Rules
1. Only helpers who have built and hosted their first mission may request uploads to this mod.
2. Do not request every music track you might use. Please make sure that what is being uploaded will be used to avoid increasing the mod size for no good reason.
3. Do not make your missions dependent on this mod updating in time.
4. Video files will require approval of the Event Team Lead(s) and will not be done often due to OGV file sizes.
5. No one in the team is required to keep updating this mod. If you are interested in helping with its maintenance, please learn how to use GitHub and follow the naming and file structure.
